About This Demo

This application provides two separate implementations of the same functionality. You can choose which implementation to use by clicking one of the buttons below.

The "Current Default Mode" indicator below shows the setting of the feature flag described in Chapter 11, which would determine which implementation handles requests in a real transformation scenario.

Legacy Implementation

This implementation has tangled dependencies and violates Clean Architecture principles:

  • Direct database access in route handlers
  • Business logic mixed with infrastructure concerns
  • No separation of concerns
Try Legacy Implementation
Clean Architecture Implementation

This implementation follows Clean Architecture principles with proper separation of concerns:

  • Domain logic isolated from infrastructure
  • Dependencies pointing inward
  • Well-defined interfaces between layers
